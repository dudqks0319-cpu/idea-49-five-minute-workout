# 5분 홈트 루틴

## 개요
- 아이디어 번호: 49
- 한 줄 컨셉: 짧은 운동 루틴 추천과 기록
- 수익모델: 구독
- MVP 난이도: 하
- 카테고리: 건강

## 기술 스택
- Flutter (Android/iOS/Web)
- Supabase (Postgres + Auth + RLS)

## 실행
```bash
flutter pub get
flutter analyze
flutter test
flutter run
```
